## LexiFill - Frontend

LexiFill is a web application that allows users to upload legal documents, fill dynamic placeholders via a conversational interface, and preview/download the completed document. This repository contains the frontend built with React, TypeScript, and Tailwind CSS.

## Features

- Upload .docx legal documents.

- Conversational UI to fill placeholders.

- Auto-scroll chat for seamless interaction.

- Preview filled document before download.

- Responsive and visually appealing interface.

# Tech Stack

- React + TypeScript

- Tailwind CSS

- React Router DOM

- Fetch API for backend communication

# Setup Instructions
1. Clone the Repository
git clone https://github.com/SaiBharathReddy/LexiFill-FrontEnd.git
cd LexiFill-FrontEnd

2. Install Dependencies
npm install

3. Run the App Locally
npm run dev

Open http://localhost:5173 in your browser.
